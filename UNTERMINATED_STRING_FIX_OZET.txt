✅ İYİLEŞTİRME 1: _is_truncated() Güçlendirildi
----------------------------------------------------------
Eski kontroller:
- String tırnak dengesi (basit)
- Parantez dengesi

Yeni kontroller:
- ✅ Escape karakterleri hariç tırnak sayımı
- ✅ Açık parantez/bracket/brace kontrolü
- ✅ Son satır yarım mı kontrolü
  * def, class, if, elif, else:, for, while
  * try:, except, finally:, with
  * return, import, from
  * Virgül, backslash, plus ile biten satırlar
- ✅ Triple quote (''' ve """) dengesi

Kod:
```python
def _is_truncated(self, code: str) -> bool:
    # Boş kod kontrolü
    if not code or not code.strip():
        return True
    
    # Escape edilmemiş tırnak sayımı
    single_quotes = code.count("'") - code.count("\\'")
    double_quotes = code.count('"') - code.count('\\"')
    
    # Açık parantez kontrolü
    open_parens = code.count("(") - code.count(")")
    
    # Son satır yarım mı?
    incomplete_endings = ["def ", "class ", "if ", ...]
    
    # Triple quote kontrolü
    triple_single = code.count("'''")
    triple_double = code.count('"""')
```

```python
def _fix_unterminated_strings(self, code: str) -> str:
    try:
        ast.parse(code)
        return code  # Zaten geçerli
    except SyntaxError as e:
        if "unterminated" not in str(e).lower():
            return code
        
        # Hatalı satırı bul ve düzelt
        error_line = e.lineno
        lines = code.splitlines()
        bad_line = lines[error_line - 1]
        
        # Tırnak ekle
        if single_count % 2 != 0:
            lines[error_line - 1] = bad_line.rstrip() + "'"
        elif double_count % 2 != 0:
            lines[error_line - 1] = bad_line.rstrip() + '"'
        
        # Doğrula
        fixed_code = "\n".join(lines)
        ast.parse(fixed_code)
        return fixed_code
```
