1. ALTERNATİF FORMAT DESTEĞİ ✅

Fonksiyon: _extract_file_blocks()

Önceki Durum:
  • Sadece [FILE:name.py]...[/FILE] formatını destekliyordu
  • Diğer formatlar parse edilemiyordu

Yeni Durum:
  • 3 farklı format desteği eklendi:
  
  Format 1: [FILE:name.py]...[/FILE]
    • Orijinal format
    • En güvenilir
    
  Format 2: ```python # name.py ... ```
    • Markdown code block + comment
    • Qwen2.5-Coder'ın kullandığı format
    
  Format 3: ```name.py ... ```
    • Markdown code block + filename
    • Alternatif format

Kod Değişikliği:
  ```python
  def _extract_file_blocks(self, text: str) -> list[dict]:
      """Extract [FILE:name.py]...[/FILE] blocks from LLM response.
      
      Supports multiple formats:
      1. [FILE:name.py]...[/FILE]
      2. ```python # name.py ... ```
      3. ```name.py ... ```
      """
      import re
      files = []
      
      # Format 1: [FILE:name.py]...[/FILE]
      pattern1 = r'\[FILE:([^\]]+)\]\n?(.*?)\[/FILE\]'
      matches1 = re.findall(pattern1, text, re.DOTALL)
      for filename, content in matches1:
          filename = filename.strip()
          content = content.strip()
          if content:
              files.append({"filename": filename, "content": content})
      
      # Format 2: ```python # name.py ... ```
      pattern2 = r'```python\s*#\s*([^\n]+)\n(.*?)```'
      matches2 = re.findall(pattern2, text, re.DOTALL)
      for filename, content in matches2:
          filename = filename.strip()
          content = content.strip()
          if content and filename:
              files.append({"filename": filename, "content": content})
      
      # Format 3: ```name.py ... ```
      pattern3 = r'```([a-zA-Z0-9_\-\.]+\.[a-z]+)\n(.*?)```'
      matches3 = re.findall(pattern3, text, re.DOTALL)
      for filename, content in matches3:
          filename = filename.strip()
          content = content.strip()
          # Sadece dosya uzantısı varsa ekle
          if content and filename and '.' in filename:
              # Zaten eklenmiş mi kontrol et
              if not any(f['filename'] == filename for f in files):
                  files.append({"filename": filename, "content": content})
      
      return files
  ```

Avantajlar:

  ✅ Qwen2.5-Coder ile uyumlu
  ✅ Geriye dönük uyumlu (eski format hala çalışıyor)
