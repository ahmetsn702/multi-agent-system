─── 3.1. MEMORY AGENT SINIFI ───

Dosya: core/memory_agent.py

Özellikler:
  • Tamamlanan projeleri JSON'a kaydeder
  • Keyword-based arama (tag matching)
  • Proje dosyalarını ve kod snippet'lerini indexler
  • Yeni görevler için ilgili projeleri bulur
  • Planner'a context sağlar

Metodlar:
  • save_project(slug, goal, result) - Proje kaydet
  • search_relevant(goal, max_results=3) - İlgili projeleri bul
  • format_context(goal) - Planner için context formatla
  • list_all() - Tüm projeleri listele
  • _extract_tags(text) - Keyword çıkar

Veri Yapısı:
  {
    "projects": {
      "slug": {
        "slug": "proje-adi",
        "goal": "Kullanıcı hedefi",
        "created_at": "2026-03-01T12:00:00",
        "files": ["src/main.py", "tests/test_main.py"],
        "snippets": {"main.py": "import flask..."},
        "cost": 0.0234,
        "tasks_completed": 5,
        "tags": ["flask", "api", "sqlite"]
      }
    }
  }

Kayıt Yeri: workspace/memory/project_memory.json

─── 3.2. ORCHESTRATOR ENTEGRASYONU ───

Dosya: core/orchestrator.py

Değişiklikler:
  1. Import eklendi:
     from core.memory_agent import get_memory_agent

  2. run() metodunun BAŞINDA (Plan öncesi):
     • Memory agent'tan ilgili projeler aranır
     • Bulunan projeler plan_input'a eklenir
     • Planner önceki benzer projeleri görerek daha iyi plan yapar

  3. run() metodunun SONUNDA (Başarılı tamamlanma):
     • Proje memory'ye kaydedilir
     • Dosya listesi, snippet'ler, cost, tags kaydedilir
     • Hata olursa sistem devam eder (try/except)

Kod Örneği:
  # Başlangıç
  memory = get_memory_agent()
  memory_context = memory.format_context(user_goal)
  plan_input += "\n" + memory_context

  # Bitiş
  memory.save_project(slug, user_goal, final_output)

───────────────────────────────────────────────────────────────────────────────
  HATA YÖNETİMİ
───────────────────────────────────────────────────────────────────────────────

Tüm memory agent çağrıları try/except ile sarılı:
  • Memory hatası olursa sistem devam eder
  • Console'a hata yazılır ama proje tamamlanır
  • Kullanıcı deneyimi etkilenmez

Örnek:
  try:
      memory = get_memory_agent()
      memory.save_project(slug, user_goal, final_output)
  except Exception as e:
      print(f"[Memory] Kayıt hatası (devam ediliyor): {e}")
