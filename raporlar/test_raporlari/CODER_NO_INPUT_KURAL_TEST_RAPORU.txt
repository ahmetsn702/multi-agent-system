🚨 KRİTİK KURAL - INPUT() YASAK:
  - ASLA input() fonksiyonu kullanma!
  - Kullanıcı girdisi gereken yerlerde sys.argv veya fonksiyon 
    parametresi kullan
  - Interaktif programlar yerine argüman tabanlı tasarım yap
  
YANLIŞ ❌:
  sayi = int(input("Sayı girin: "))
  
DOĞRU ✅:
  import sys
  sayi = int(sys.argv[1])
  
VEYA:
  def topla(a: int, b: int) -> int:
      return a + b
  
  if __name__ == "__main__":
      import sys
      a = int(sys.argv[1])
      b = int(sys.argv[2])
      print(topla(a, b))
