YENİ DOSYA: agents/ui_tester_agent.py (350+ satır)
- UITesterAgent sınıfı (BaseAgent'tan türetildi)
- 3 test modu:
  1. Flask app: server başlat → screenshot → durdur
  2. FastAPI app: server başlat → screenshot → durdur

ENTEGRASYON:

- critic_agent.py → act() güncellendi
  * Screenshot varsa 6. kriter ekler: ui_quality
  * System prompt güncellendi
