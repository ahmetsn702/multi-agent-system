✅ GELİŞTİRME 1: Critic → Coder Feedback Loop
═══════════════════════════════════════════════════════════════════════════════

Dosya: core/orchestrator.py
Fonksiyon: _critic_review()

Yapılan Değişiklikler:
  1. Maksimum iterasyon sayısı: 3 (sonsuz döngü önleme)
  2. Her iterasyonda Critic feedback'i Coder'a iletiliyor
  3. "approve" gelirse döngüden çıkılıyor
  4. İterasyon sayısı loglanıyor: "[Orchestrator] Iterasyon 2/3"
  5. 3 iterasyon sonunda hala "revise" ise devam ediliyor (takılı kalma)

Yeni Özellikler:
  • max_iterations = 3 değişkeni eklendi
  • approved flag kontrolü eklendi
  • Detaylı feedback mesajı (skor, özet, sorunlar, iyileştirmeler)
  • critic_feedback context'e eklendi
  • İterasyon sayısı metadata'ya eklendi
  • MAX_ITERATIONS_REACHED verdict'i eklendi

Beklenen Akış:
  Coder → Critic (revise) → Coder → Critic (revise) → Coder → Critic (approve) → Executor

Loglar:
  • "[Orchestrator] Iterasyon 1/3"
  • "[Orchestrator] 🔄 Revizyon gerekli (Skor: 6.5/10) - Coder'a geri gönderiliyor..."
  • "[Orchestrator] ✅ Critic onayladı (2 iterasyon)"
  • "[Orchestrator] ⚠️ Maksimum iterasyon (3) aşıldı, devam ediliyor..."

✅ GELİŞTİRME 2: Executor Hata Analizi
═══════════════════════════════════════════════════════════════════════════════

Dosya: agents/executor_agent.py
Fonksiyonlar: act(), _parse_error()

Yapılan Değişiklikler:
  1. Hata mesajları parse ediliyor
  2. Parse edilmiş feedback Coder'a iletiliyor
  3. Hata detayları loglanıyor

Yeni Fonksiyon: _parse_error()
  Parse edilecek hata tipleri:
    • ImportError / ModuleNotFoundError
    • SyntaxError
    • AttributeError
    • FileNotFoundError
    • IndentationError
    • NameError
    • TypeError
    • Genel Exception

Çıktı Formatı:
  {
    "status": "failed",
    "error_type": "ImportError",
    "error_message": "No module named 'flask'",
    "parsed_feedback": "Eksik paket tespit edildi: flask. Coder bu import'u kaldırmalı veya requirements.txt'e eklemeli.",
    "raw_output": "...",
    "suggested_fix": "pip install flask"
  }

Örnekler:
  • ImportError: "Eksik paket: flask, pip install gerekli"
  • SyntaxError: "Satır 15'de syntax hatası: parantez eksik olabilir"
  • AttributeError: "'str' nesnesinde 'append' özelliği yok"
  • FileNotFoundError: "Dosya bulunamadı: data.txt"

Loglar:
  • "[Executor] ❌ Hata tespit edildi: ImportError → Coder'a iletiliyor"
  • "[Executor] ❌ Shell hatası: SyntaxError"

Entegrasyon:
  • act() fonksiyonunda hata kontrolü eklendi
  • error_analysis results'a eklendi
  • Shell komut hataları da parse ediliyor
