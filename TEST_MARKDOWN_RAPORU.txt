ÖNERİ 1: REGEX-BASED POST-PROCESSING (ÖNEMLİ) ⭐⭐⭐
  Dosya: coder_agent.py
  Metod: _check_syntax()
  
  LLM'e güvenmek yerine regex ile otomatik düzelt:
  
  ```python
  import re
  
  # if/while/for ifadelerinde eksik değerleri düzelt
  code = re.sub(r'if\s+(\w+)\s*==\s*:', r'if \1 == 0:', code)
  code = re.sub(r'if\s+(\w+)\s*!=\s*:', r'if \1 != 0:', code)
  
  # sys.path.insert eksik parametreyi düzelt
  code = re.sub(r'sys\.path\.insert\(\s*,', r'sys.path.insert(0,', code)
  
  # Genel: == : veya != : gibi hataları düzelt
  code = re.sub(r'(==|!=|<|>|<=|>=)\s*:', r'\1 0:', code)
  ```
  
  Neden: LLM güvenilir değil, regex kesin çözüm
