====================================================================
  KARŞILAŞTIRMA: ESKI vs YENİ TASARIM
====================================================================

ESKI TASARIM (input() ile):
  ❌ username = input("Kullanıcı adı: ")
  ❌ password = input("Şifre: ")
  ❌ CLI tabanlı interaktif program
  ❌ Executor takılır
  ❌ Otomatik test edilemez

YENİ TASARIM (REST API ile):
  ✅ @app.route("/register", methods=["POST"])
  ✅ data = request.json
  ✅ username = data.get("username")
  ✅ Web API tabanlı
  ✅ Executor uyumlu
  ✅ Otomatik test edilebilir (pytest + requests)
