1. GENEL BAKIŞ
--------------
Proje, birden fazla yapay zeka ajanının bir araya gelerek yazılım
geliştirme görevlerini otonom olarak tamamlamasını sağlayan bir
orkestrasyon sistemidir.

Kullanıcı bir doğal dil komutu girer (ör. "Python ile hesap
makinesi yaz"), sistem bunu otomatik olarak planlar, araştırır,
kodlar, test eder ve Git'e kaydeder.

4. BU OTURUMDA YAPILAN DÜZELTMELER
------------------------------------
  [x] Critic Ajanı — hep 5/10 verme sorunu çözüldü
      Neden: gpt-oss-120b JSON parse edilemiyordu, score 0'a düşüp
             floor mekanizması 5.0 atıyordu.
      Çözüm: JSON parse başarısızsa otomatik retry eklendi.

  [x] Dosya Sayacı — "toplam dosya: 0" sorunu çözüldü
      Neden: Orchestrator yanlış dict path'inde saved_files arıyordu.
      Çözüm: 3 farklı path'te arama yapacak şekilde güncellendi.

  [x] Import Yolu — "ModuleNotFoundError: No module named 'src'"
      Neden: Coder, modülleri "src.modul" şeklinde import ediyordu.
      Çözüm: Coder promptuna IMPORT KURALLARI bölümü eklendi:
             - main.py her zaman sys.path.insert ile başlamalı
             - ASLA "src.modul" yazma, doğrudan "modul" yaz

  [x] JSON Metin → .py Sorunu — output.py'nin içi JSON çıkıyordu
      Neden: Fallback kod, [FILE:] bloğu bulunamazsa tüm LLM
             çıktısını (JSON dahil) dosyaya yazıyordu.
      Çözüm: Tehlikeli fallback kaldırıldı. Şimdi [FILE:] bloğu
             bulunamazsa → LLM'e retry yapılıyor, sıkı prompt ile.

  [x] Coder Model Değişikliği
      Eski: Qwen3 Coder Next (80B total, 3B aktif, $0.12/$0.75)
      Yeni: Grok 4 Fast (parametre gizli, 2M context, $0.20/$0.50)
      Fark: Output %33 ucuzladı, context 262K → 2.000.000 token
