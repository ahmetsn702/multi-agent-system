═══════════════════════════════════════════════════════════════
  REGEX PATTERN'LER
═══════════════════════════════════════════════════════════════

1. if/while + eksik değer:
   Pattern: r'(if|while)\s+(\w+)\s*(==|!=|<|>|<=|>=)\s*:'
   Örnek: if b == : → if b == 0:

2. sys.path.insert eksik parametre:
   Pattern: r'sys\.path\.insert\s*\(\s*,'
   Örnek: sys.path.insert(, ...) → sys.path.insert(0, ...)

3. Boş karşılaştırma:
   Pattern: r'(==|!=)\s*:'
   Örnek: == : → == 0:
